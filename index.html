Writing negroni middleware in golang

Negroni is a great middleware framework for golang that comes with a lot of functionallity (logging, panic recovery, cors) that one needs to create for almost each web application. It is very straightforward to use negroni throughout the development phase of a project, then replace it with custom code in production.
